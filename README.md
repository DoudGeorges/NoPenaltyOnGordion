# NoPenaltyOnGordion

A [BepInEx](https://github.com/BepInEx/BepInEx) plugin for [Lethal Company](https://store.steampowered.com/app/1966720/Lethal_Company/) with 21,500+ downloads on [Thunderstore](https://thunderstore.io/c/lethal-company/p/DoudGeorges/NoPenaltyOnGordion/) that removes the credit penalty for player deaths on the Company moon (Gordion).

> **Note:** This project is no longer actively maintained and has been archived. No further updates, bug fixes, or pull requests will be accepted.
